# A tcp api to download picture with painter's id at pixiv.

## ⚙ Building
### Dependent installation
```shell
npm install
```
or
```shell
yarn install
```

### Compile the software
```shell
npm run build
```
or
```shell
yarn run build
```

## 🏃 Running
```shell
npm run dev
```
or
```
yarn run dev
```
